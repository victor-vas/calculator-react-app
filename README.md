## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/victor-vas/calculator-react-app calculator-react-app

# Go into the repository
$ cd calculator-react-app

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
